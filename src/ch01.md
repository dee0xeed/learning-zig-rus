
# Обзор языка. Часть I

Вот так выглядит исходный код на языке Zig:

```zig
const std = @import("std");

// Если убрать слово 'pub', то этот код не откомпилируется
pub fn main() void {
    const user = User{
        .power = 9001,
        .name = "Goku",
    };

    std.debug.print("{s}'s power is {d}\n", .{user.name, user.power});
}

pub const User = struct {
    power: u64,
    name: []const u8,
};
```
